# 简单小知识，让你ppt专业很多

今天跟让设计MM帮着我调整了一下技术架构图的色调，学习了很多东西，按着她说的配色之后确实看着专业与舒服些了！

关于配色常识：打开配色版

- 横着一行或者竖着一列按顺序选；
- 如果颜色个数比较少，可以挑着按顺序选；
- 如果对比强烈可以按对角线选择；
- 一般红色橙色两列，蓝色两列用的人会比较多；
- 另外打开调色板，30度距离是比较和谐的，45度是安全的
  - 尽量不要用对比强烈的颜色，专业设计师用，新手掌控不了
    - 比如再一片红色中，用一个绿色箭头，通过面积比保证和谐程度
  - 淡绿、淡黄色，春天的颜色，小清新，治愈系
  - 蓝色区域，商务、科技风
  - 苹果活了之后流程大片的银色灰色背景，带几个字
